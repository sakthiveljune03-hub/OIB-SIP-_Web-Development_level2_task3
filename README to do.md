# To-Do Web App

## Overview
This To-Do Web App is a simple application designed to help users manage their daily tasks. The app allows users to add, delete, and mark tasks as complete. It features a clean interface that separates pending and completed tasks.

## Features
- ✅ **Add Tasks**: Enter new tasks using the input field and click "Add"
- ✅ **Complete Tasks**: Mark tasks as complete with one click
- ✅ **Delete Tasks**: Remove tasks from both pending and completed lists
- ✅ **Task Organization**: Separate display for pending and completed tasks
- ✅ **User-Friendly Interface**: Simple and intuitive design

## Technologies Used
- **HTML5**: For structuring the web application
- **CSS**: For styling and layout (can be enhanced)
- **JavaScript**: For interactive functionality and task management
- **DOM Manipulation**: For dynamically updating the task lists
