# To-Do Web App

## Overview
This To-Do Web App is a simple application designed to help users manage their daily tasks. The app allows users to add, edit, delete, and check off tasks as they complete them. It is built using HTML, CSS, and JavaScript.

## Features
- **Add Tasks**: Users can add new tasks to their to-do list.
- **Edit Tasks**: Users can edit existing tasks to update their details.
- **Delete Tasks**: Users can remove tasks that are no longer relevant.
- **Mark as Complete**: Users can check off tasks that they have completed.
- **Local Storage**: Tasks are stored in the browser's local storage, so they persist even after refreshing the page.

## Technologies Used
- **HTML**: For structuring the web app.
- **CSS**: For styling the app and making it visually appealing.
- **JavaScript**: For adding interactivity and managing tasks.
- **Local Storage**: For storing tasks in the user's browser.

## Usage
- To add a task, enter the task description in the input field and press the Add button.
- To edit a task, click on the task description, make your changes, and press Enter.
- To delete a task, click on the delete icon next to the task.
- To mark a task as complete, click on the checkbox next to the task.
